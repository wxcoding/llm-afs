# 基于大模型的防诈骗知识问答系统 - 架构设计

## 1. 系统概述

本系统是一个基于大模型的防诈骗知识问答平台，帮助用户了解常见诈骗手法、提高防骗意识。

## 2. 技术栈

### 后端
- Java 17
- Spring Boot 3.2
- MyBatis Plus
- MySQL 8.0
- Lombok
- OkHttp (调用大模型API)

### 前端
- Vue 3.4
- Element Plus 2.5
- Vite 5.0
- Axios
- Vue Router 4

### 大模型
- 通义千问API (qwen-turbo)
- 支持扩展其他大模型

## 4. 功能模块

### 4.1 用户模块
- 用户注册/登录
- 会话管理

### 4.2 问答模块
- 智能问答（基于大模型）
- 对话历史记录
- 上下文理解

### 4.3 案例模块
- 典型诈骗案例展示
- 案例分类（电信诈骗、网络诈骗、冒充诈骗等）

### 4.4 知识库模块
- 防诈骗知识文章
- 常见问题FAQ
- 防范技巧

## 5. 数据库设计

### 用户表 (user)
- id, username, password, create_time

### 对话表 (chat_session)
- id, user_id, title, create_time

### 消息表 (chat_message)
- id, session_id, role, content, create_time

### 案例表 (scam_case)
- id, title, type, content, create_time

### 知识库表 (knowledge)
- id, title, category, content, create_time

## 6. API设计

### 用户接口
- POST /api/user/register - 注册
- POST /api/user/login - 登录

### 问答接口
- POST /api/chat/send - 发送消息
- GET /api/chat/history/{sessionId} - 获取历史

### 案例接口
- GET /api/cases - 案例列表
- GET /api/cases/{id} - 案例详情

### 知识库接口
- GET /api/knowledge - 知识列表
- GET /api/knowledge/{id} - 知识详情
